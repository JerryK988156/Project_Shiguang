## 二、实现清单

### 2.1 用户模块（100%）

| 功能 | 说明 |
| --- | --- |
| 登录 / 退出 | Session 鉴权，禁用账号检查 |
| 获取当前用户信息 | 含 id/username/nickname/role/avatar |

## 三、技术实现亮点

| 亮点 | 说明 |
| --- | --- |
| 三级权限体系 | 前端 store + 路由守卫 + 后端 SessionUtils，超规划实现 |
| 头像完整上传链路 | 文件校验 → 本地存储 → 静态资源映射 → Pinia 即时同步 |
| 数据库自检 | 启动时自动验证连接 + 一键 `force=true` 重建 |
| 日历热力图 | ECharts calendar heatmap，按日聚合，深绿浅绿区分 |
| Session 鉴权 | 所有接口均经过 `SessionUtils.requireUserId/requireAdmin` |
| utf8mb4 | DDL 显式指定 + Hikari connection-init-sql + 自定义 SQL 执行器 |
| Vue 响应式刷新 | watch route + onMounted 双重保障，切换页面对图表同步更新 |
